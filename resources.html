Part 2

As a part of the project, I think I have really improved my overall skills on Research, Photoshop and HTML. I feel confident about HTML and would continue to experiment in Photoshop. I had keen interest in this topic for a long and time and I tried to explain the topic in simple words. While doing this project I created the background from an image and added an opaque layer under the text using Photoshop. So that it is easy to read the text.
I felt proud of myself while doing the CSS image rollover and using a button on top of that. It really boosted my confidence in CSS which I lacked. I think I have done my best in creating the logo, using CSS image rollover and Content.
I could have improved my Responsive Designs, Colours, CRAP, but I am really lack the knowledge of colour combinations. I would surely try to improve that over the period. Overall this project was fun to do, and I enjoyed doing it. Thank You!
